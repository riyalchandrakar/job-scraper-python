# Job Listings Web Scraping & Analysis (Python)

This project demonstrates web scraping and basic data analysis using Python.

## Features
- Web scraping using BeautifulSoup
- Handling missing and dynamic fields
- Data cleaning and preprocessing
- CSV export and basic analysis

## Tech Stack
- Python
- Requests
- BeautifulSoup
- Pandas

## How to Run
```bash
pip install -r requirements.txt
python scraper.py
